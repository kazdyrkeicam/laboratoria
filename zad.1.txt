Funkcja powinna sprawdziæ, za pomoc¹ operacji bitowych, czy cyfry przekazanej liczby ca³kowitej s¹ parzyste.
Ponadto funkcja powinna ustawiæ przekazany wskaŸnik na bool na wartoœæ true,

#include <stdio.h>
#include <stdbool.h>

void *f(int number, bool *result){
    while(number){
        if(number % 10 & 1){
            *result = false;
            return;
        }
        number /= 10;
    }
    *result = true;
}

int main()
{
    bool a;
    f(22222, &a);
    printf("%d\n", a);
    return 0;
}
